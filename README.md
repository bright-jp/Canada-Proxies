## FAQ

### カナダプロキシサーバーとは何ですか？
カナダプロキシサーバーは、カナダに拠点を置くサーバーであり、お客様のデバイスとインターネットの間の仲介役として機能します。匿名性を提供し、制限やブロックの回避を支援し、Webコンテンツのスクレイピングを可能にし、コンテンツフィルタリングポリシーを適用します。

### カナダ全域でZIPコードをターゲティングできますか？
はい、カナダのZIPコードレベルのターゲティングを使用してIPを選択できます。Bright Dataは、都市および州レベルのプロキシターゲティングも提供しています。

### 利用可能なプランの種類は何ですか？ 
Bright Dataは、以下を含む柔軟な料金モデルを提供しています。

- **Pay-As-You-Go**: GBあたりの固定料金です。
- **Subscription Plans**: 月次、年次、カスタムのオプションです。

### Bright Dataのカナダプロキシはコンプライアンスに準拠しており、安全に使用できますか？
Bright Dataのプロキシは倫理的に調達されており、GDPRおよびCCPAを含む関連するすべてのデータ保護法に準拠しています。

### 専任のサポートは利用できますか？
専任サポートチームが24時間365日、お客様を支援いたします。お問い合わせいただき、お客様のニーズについてご相談のうえ、Dedicated proxy networkの利点を最大化してください。
